# solidity-hello-world

## Prereqs

### Node 8.5
```
curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
sudo apt-get install -y nodejs
```

### TestRPC
Attach to test blockchain, seperate shell needed
```
npm install -g ethereumjs-testrpc

testrpc
```


## Install
```
npm install 
npm start
```
